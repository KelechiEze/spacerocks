Welcome to GitHub Learning!

GitHub is a platform that helps you store, track, and collaborate on code projects. It uses Git — a version control system — to keep track of changes made to your files.

What is GitHub?

GitHub is like a social media site for developers. You can share your code (called repositories), work together on projects, and keep track of changes.
